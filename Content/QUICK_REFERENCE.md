## 🔧 The 8-Step Workflow (30 sec Summary)

1. **Source** → Where is content coming from? (file/Google Doc/one-off)
2. **Brand** → OutAtlas or Travel in Pride?
3. **Platforms** → TikTok, Instagram, YouTube, Pinterest, LinkedIn, Reddit?
4. **Voiceover** → Any posts need scripts? What tone?
5. **Write** → Skill generates all captions + voiceover
6. **Review** → Approve, adjust, or rewrite
7. **Export** → Get branded .docx file
8. **Notion** *(optional)* → Add tasks to dashboard

## ✨ Pro Tips

**Tip 1: Be Specific**
- Good: "Barcelona hotel discovery, OutAtlas, TikTok + Instagram Reels, build awareness + signups"
- Bad: "Write captions"

**Tip 2: Batch Content**
- One session can handle 10+ posts
- Multi-platform batching saves hours

**Tip 3: Review the Post List**
- Always confirm platforms before writing captions
- Easier to adjust upfront than after

**Tip 4: Use Voiceovers**
- Energetic tone for circuit parties / events
- Warm tone for travel tips / discoveries
- Newsy tone for breaking news / announcements

**Tip 5: Export to .docx**
- Easy copy-paste to each platform
- Keep for records / approval
- Professional presentation for teams

## 🚫 Common Mistakes to Avoid

❌ **Don't:** Ask for one caption for multiple platforms
✅ **Do:** Specify each platform separately (skill will write different captions)

❌ **Don't:** Forget to mention the goal (awareness, signups, engagement)
✅ **Do:** Be clear about what you want the caption to achieve

❌ **Don't:** Request Instagram captions with TikTok hashtag strategy
✅ **Do:** Let the skill handle platform best practices

❌ **Don't:** Skip the review step
✅ **Do:** Approve post list and tone before full writing

## 🆘 Troubleshooting 30-Seconds

**"Captions feel off-brand"**
→ Specify the tone more clearly (e.g., "more playful" or "more professional")

**"Hashtags are wrong"**
→ The skill uses platform best practice counts. You can adjust them in the review step.

**"I need a different angle"**
→ Ask it to "rewrite with a [different angle]" — e.g., "more personal" or "more professional"

**"Can't find the .docx"**
→ It's saved to the output location. Check the export confirmation message for the path.

**"Voiceover feels rushed"**
→ Request "slower pace" or "add more pauses" in the delivery notes
