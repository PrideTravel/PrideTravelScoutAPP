# .DOCX Templates Guide

## Overview

Since the skills can't be directly modified (read-only), here's the practical solution:

**Each skill should produce a `.docx` file with the exact structure shown below.** You can use these templates as examples, or I can help you integrate the docx skill into each skills workflow.

## 1. CONTENT-PLANNER .DOCX TEMPLATE

**Filename:** `[Brand]_ContentCalendar_[DateRange].docx`
**Example:** `OutAtlas_ContentCalendar_Mar28-31.docx`

### Structure:

```
PAGE 1:
├─ Title: "Content Calendar — [Brand Name]"
├─ Subtitle: "[Date Range]" (italics)
│
├─ Section: "Executive Summary"
│  └─ 1-2 sentences: total posts, platforms, theme, assets needed
│
├─ Section: "Content Calendar Overview"
│  └─ TABLE with columns: Date | Platform | Concept | Status | Asset
│     └─ Rows for each post with: date, platform, 1-line concept, approval status, asset type
│
├─ Section: "Detailed Post Breakdown"
│  └─ For EACH post:
│     ├─ Post Title (Heading 3)
│     ├─ Concept: [one sentence]
│     ├─ Hook: "[exact text]"
│     ├─ Caption: "[exact text]"
│     ├─ CTA: "[primary and secondary]"
│     ├─ Hashtags: [list]
│     ├─ Platform-specific versions (TikTok, Instagram, YouTube, etc.)
│     ├─ Text on screen (timed overlays)
│     └─ Asset guidance

PAGE 2+:
├─ [PAGE BREAK]
├─ Section: "Historical Record"
│  ├─ Batch Date: [date]
│  ├─ Brand: [OutAtlas / Travel in Pride]
│  ├─ Theme: [theme name]
│  ├─ Posts Planned: [#]
│  ├─ Posts Approved: [#]
│  ├─ Posts in Revision: [#]
│  ├─ Status Summary: [1-2 sentences]
│  └─ Week 3–5 Note: [strategic notes for ads campaign]
```

### Example Content:

**Title Page:**
```
Content Calendar — OutAtlas
March 28–31, 2026

Executive Summary
3 posts planned across TikTok, Instagram Reels, and YouTube Shorts. Theme: "Hidden Gems in Spanish Cities". All posts approved and ready for asset capture. AI-generated imagery recommended for visual consistency.
```

| Sat 3/29 | Instagram Reels | Madrid Rooftop Bars | ✅ Approved | AI Video |

## HOW TO IMPLEMENT THIS

### Option A: Manual Creation
1. Use Microsoft Word or Google Docs
2. Copy the structure above
3. Fill in your specific content
4. Save as `.docx` file with naming convention: `[Brand]_[Type]_[DateRange].docx`

### Option B: Automate with Python
Use `python-docx` library:
```python
from docx import Document
from docx.shared import Pt, RGBColor

doc = Document()
doc.add_heading('Content Calendar — OutAtlas', 0)
doc.add_paragraph('March 28–31, 2026').runs[0].italic = True
# ... add more content

doc.save('OutAtlas_ContentCalendar_Mar28-31.docx')
```

### Option C: Use the docx Skill
Once these templates exist, the three skills (content-planner, growth-traffic-manager, professional-blog-auditor) can reference them and invoke the `/docx` skill to generate formatted output automatically.
