# Caption & Voiceover Writer for OutAtlas & Travel in Pride

## Overview

This skill transforms content ideas into platform-specific captions, hashtags, CTAs, and voiceover scripts optimized for maximum engagement. It handles everything from single urgent posts to multi-post content calendars with diverse platform requirements.

---

## Workflow: 8 Steps

### STEP 1 — Identify the Source

Start by asking all questions at once (don't drip questions one at a time):

```
Let's write your captions! First, a few quick questions:

1. Where is the content coming from?
   - From a recent content-planner session (I'll pull those posts)
   - I'm uploading a content calendar document (.docx, .csv, .xlsx)
   - I'm sharing a Google Doc link
   - This is a one-off / urgent post (not on the calendar)

2. Which brand?
   - OutAtlas
   - Travel in Pride
   - Both

3. Which platforms are we writing captions for?
   - TikTok
   - Instagram Reels
   - Instagram Feed (carousel or single image)
   - Instagram Stories
   - YouTube Shorts
   - YouTube Long-form
   - Pinterest
   - LinkedIn
   - Reddit
   - All platforms listed in the content plan
```

### STEP 2 — Handle One-Off / Urgent Posts

If the user selects "one-off / urgent post", ask:

```
Got it — let's get this posted fast. Tell me:

1. What happened / what is this post about?
   (e.g., "I'm in Barcelona and found an amazing queer bar",
    "LGBTQ+ travel ban news just dropped",
    "OutAtlas just hit 1,000 waitlist signups")

2. What platforms is this going on?

3. Do you have an asset (photo/video) or is this text-only?

4. What's the goal?
   - Build awareness
   - Drive app signups / waitlist
   - Community engagement
   - Share news / commentary
   - Celebrate a milestone

5. Any specific angle or message you want to hit?
```

Treat this as a single post entry and proceed to Step 4.

### STEP 3 — Pull & Confirm the Post List

**If from content-planner:** Summarize the posts from the session and display them as a numbered list for confirmation:

```
Here are the posts from your content plan. Confirm this is correct before I write captions:

1. [Post concept] — [Platform(s)] — [Date]
2. [Post concept] — [Platform(s)] — [Date]
...

Add, remove, or adjust anything before I start writing?
```

**If from uploaded file or Google Doc:**
- Read the file/doc
- Extract all posts, dates, platforms, and any existing notes
- Display the same numbered confirmation list
- Note: the output .docx will be named based on the uploaded filename

**If posts are being cross-posted to multiple platforms:**
Note in the confirmation list which posts require multiple caption versions (e.g., "Post 3 — TikTok + Instagram + LinkedIn = 3 captions").

### STEP 4 — Voiceover Check

After the post list is confirmed, ask:

```
Do any of these posts need a voiceover script?
(Select all that apply, or say "none")
[List each post by number]

For any posts that need a voiceover, what tone of voice should I use?
- 🟢 Friendly & warm — like talking to a gay best friend who travels
- 📘 Informative & authoritative — knowledgeable travel expert
- 🎉 Hype & energetic — high-energy, festival/event vibe
- 🧘 Calm & aspirational — dreamy, wanderlust-inducing
- 🔥 Bold & provocative — punchy, opinion-led, stops the scroll
- 📰 Newsy & direct — breaking news / timely commentary tone

You can assign a different tone per post, or one tone for all.
```

### STEP 5 — Write All Captions & Voiceovers

For EACH post, write a complete caption package. Use this structure:

### Platform Captions

Write a separate caption block for EVERY platform this post is going to. Never merge platforms. If a post is going to 4 platforms, write 4 distinct captions.

#### 🎵 TIKTOK

**Best Practices:**
- Caption limit: 2,200 chars (but only ~150 shown before "more" — lead with the hook)
- No intro like "hey guys" or title card text — hook must be in first line
- Hashtags: 3–5 max (quality over quantity)
- CTA drives comments or follows, NOT external links (TikTok suppresses link-heavy captions)
- Best post times: Tue/Thu/Sat 7–9pm

```
[CAPTION TEXT — conversational, punchy, emoji-forward]
[HASHTAGS — max 5, mix of niche + broad]
[CTA — e.g., "Comment your dream destination 👇" or "Follow for more hidden gems ✈️"]
```

#### 📸 INSTAGRAM REELS

**Best Practices:**
- Caption limit: 2,200 chars (125 shown before "more" — lead with the hook)
- Hashtags: 8–15 (mix of niche, location, broad)
- CTA can include "link in bio"
- Emojis used intentionally, not decoratively
- Best post times: M/W/F 11am–1pm or 7–9pm

```
[CAPTION TEXT — slightly longer than TikTok, personality-forward]
[HASHTAGS — 8–15, separated from main caption]
[CTA — e.g., "Save this for your next trip 🗺️ Link in bio to join the waitlist"]
```

#### 🖼️ INSTAGRAM FEED (carousel or single image)

**Best Practices:**
- Can be longer than Reels — feed posts support storytelling
- First line is the hook
- Use line breaks for readability
- Same hashtag strategy as Reels (8–15)
- Carousel: first slide caption should tease "swipe for [X]"

```
[CAPTION TEXT]
[HASHTAGS]
[CTA]
```

#### 📱 INSTAGRAM STORIES

**Best Practices:**
- Stories are interactive — suggest a sticker (poll, question, link, countdown)
- Text on-screen should be max 1–2 lines
- Caption is minimal — the visual + sticker does the work
- Stories disappear in 24hrs — use for urgent/timely content first

```
SUGGESTED TEXT ON SCREEN: "[Short, bold overlay text]"
STICKER: [Poll / Question / Link / Countdown — with suggested text]
SWIPE UP / LINK STICKER CTA: "[CTA text]"
```

#### ▶️ YOUTUBE SHORTS

**Best Practices:**
- Title: 60 chars max, curiosity-gap or keyword-rich, sentence case
- Description: 2–3 sentences with primary keyword in first line, link to waitlist/site
- Tags: 5–8 relevant tags
- No hashtags in title — hashtags in description only (max 3)

```
TITLE: [60 chars max — SEO + curiosity]

DESCRIPTION:
[Sentence 1 — primary keyword, what the video is about]
[Sentence 2 — value or hook for the viewer]
[Sentence 3 — CTA + link]

#[Hashtag1] #[Hashtag2] #[Hashtag3]

TAGS: [tag1], [tag2], [tag3], [tag4], [tag5]
```

#### 🎬 YOUTUBE LONG-FORM

**Best Practices:**
- Title: 60–70 chars, SEO-optimized, front-loaded keyword
- Description: 3-part structure (hook paragraph → chapters/timestamps → links + keywords)
- Tags: 8–12 tags
- Pinned comment suggestion included

```
TITLE: [SEO-optimized, 60–70 chars]

DESCRIPTION:
[Hook paragraph — 2–3 sentences, primary keywords in sentence 1]

[Chapters — if video is 5+ min:]
00:00 — Intro
[XX:XX] — [Section]
[XX:XX] — [Section]

[Links section:]
🌐 Download OutAtlas: [link]
📩 Join the waitlist: [link]
📸 Follow on Instagram: [link]

[Keyword-rich paragraph — 3–4 sentences describing the video for SEO]

TAGS: [8–12 tags]

PINNED COMMENT SUGGESTION:
"[Engaging question or CTA to drive early engagement]"
```

#### 📌 PINTEREST

**Best Practices:**
- Pin title: 75–100 chars, keyword-rich, descriptive (Pinterest is a search engine)
- Description: 150–300 chars, 2–3 sentences, naturally includes keywords
- No exclamation marks or emoji overload — Pinterest users are in planning mode
- 5–10 hashtags, all keyword-focused
- Board suggestion included
- Best days: Sat/Sun evenings

```
PIN TITLE: [75–100 chars, keyword-rich, sentence case]

DESCRIPTION:
[Sentence 1 — what this pin is about, includes primary keyword]
[Sentence 2 — value for the pinner / what they'll learn or discover]
[Sentence 3 — CTA]

#[keyword1] #[keyword2] #[keyword3] #[keyword4] #[keyword5]

SUGGESTED BOARD: [e.g., "LGBTQ+ Travel Destinations" or "Gay Travel Tips 2026"]
```

#### 💼 LINKEDIN

**Best Practices:**
- Opens with a bold hook sentence (no preamble)
- Personal, insight-led voice — not promotional
- Short paragraphs (1–2 sentences each) for mobile readability
- "See more" triggers at ~210 chars — put the most compelling content before that
- 3–5 hashtags max, professional/industry-relevant
- CTA leads to article, app, or waitlist — not "follow me"
- Post as Drew Lewis personally OR as Travel in Pride page (note which)
- Best days: Tue–Thu, 8–10am

```
[Hook line — bold statement, question, or insight]

[Paragraph 1 — context or personal story]

[Paragraph 2 — insight or data point]

[Paragraph 3 — what this means for LGBTQ+ travelers / the industry]

[CTA — e.g., "OutAtlas is building the solution. Join the waitlist at OutAtlas.com"]

#[Hashtag1] #[Hashtag2] #[Hashtag3]

POST AS: [Drew Lewis personally / Travel in Pride page]
```

#### 👾 REDDIT

**Best Practices:**
- NO self-promo tone whatsoever — Reddit will downvote/remove it
- Lead with genuine value, question, or discussion starter
- Mention OutAtlas/Travel in Pride only if it's relevant and natural
- No hashtags
- Post as text post or link post (note which)
- Subreddit recommendations included

```
SUBREDDITS: [2–3 recommended subreddits]

POST TYPE: [Text / Link / Image]

TITLE: [Curiosity or community-focused — not branded]

BODY:
[Opening — genuine observation, story, or question]
[Middle — value, context, or discussion prompt]
[Optional closing — natural mention of OutAtlas if appropriate, framed as a resource not an ad]

FLAIR: [Suggested flair if applicable]
```

## Key Principles

1. **Platform Compliance First** — Never merge platforms. Each platform gets its own caption optimized for that platform's algorithm, culture, and user behavior.

2. **Hook Before Truncation** — Always put the most compelling content before "more" truncation points. The first 125–150 chars are everything.

3. **CTA Drives the Goal** — CTAs should align with the post's goal (awareness, signups, engagement, shares). Don't waste the CTA.

4. **Brand Voice Consistency** — Both brands have distinct voices. Maintain consistency across all platforms and posts.

5. **Hashtag Strategy** — Mix niche hashtags (highly specific, lower volume) with broad hashtags (high volume, more competition). Never hashtag-spam.

6. **Voiceovers Sound Natural** — Write scripts as if speaking to a friend, not reading from a script. Use pauses, emphasis, and natural rhythm.
