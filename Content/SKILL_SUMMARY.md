## 📱 Platform Best Practices Included

**TikTok**
- Hook in first line (150 chars before truncation)
- 3-5 hashtags max
- Engagement-driven CTAs (no external links)
- Best times: Tue/Thu/Sat 7-9pm

**Instagram Reels**
- Hook before 125-char truncation
- 8-15 hashtags
- "Link in bio" CTA
- Best times: M/W/F 11am-1pm or 7-9pm

**YouTube Shorts**
- SEO-optimized title (60 chars)
- Keyword in first sentence
- 3 hashtags only
- Tags for recommendations

**YouTube Long-form**
- Chapters with timestamps
- Keyword-rich description
- Hook + links structure
- 8-12 tags for recommendations

**Pinterest**
- Keyword-rich title (75-100 chars)
- Natural keyword placement (no spam)
- Description 150-300 chars
- Best days: Sat/Sun evenings

**LinkedIn**
- Hook sentence first
- ~210 chars before "see more"
- Professional 3-5 hashtags
- Best times: Tue-Thu 8-10am

**Reddit**
- Zero self-promo tone
- Community-first approach
- Genuine value-led
- Brand mention only if natural
